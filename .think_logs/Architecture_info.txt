3. Jetson Transmission
No protobufs used. Transmission is pure JSON over a WebRTC RTCDataChannel named nav_meta.

Send location: services/edge_gateway/app.py:309-310


data_channel.send(orjson.dumps(metadata).decode("utf-8"))
Full metadata dict is sent (all fields from build_metadata() — see Section 2). The Jetson receives it at jetson_client/app.py @metadata_channel.on("message") and logs two events: metadata_rx (full JSON string) and inference_timing_rx (extracted timing fields only).

4. Log Mapping & Output Fields
Data Flow Table
Source Component	Field/Data	Destination
inference_loop()	Full metadata dict	logs/edge_session.jsonl (event: inference_done)
inference_loop()	Full metadata dict	RTCDataChannel → Jetson
visualization_dump.py	frame_bgr + masks + depth_map	logs/visualization_artifacts/{trace_id}.npz
Jetson on_message()	Raw JSON string + rx_time_ms	logs/jetson_session.jsonl (event: metadata_rx)
Jetson on_message()	10 extracted timing fields	logs/jetson_session.jsonl (event: inference_timing_rx)
Critical Discrepancies: Runtime vs. Logged
Field	Runtime Value	JSONL Log	NPZ File
Per-object mask (bool 640×640)	Computed in decode_yolo_segmentation()	Stripped (compact YOLO objects built at app.py:239-248)	Saved as yolo_masks
depth_map (float32 HxW)	Full float array	Stripped — only percentiles logged	Saved as depth_map float16
age_ms	inference_ts_ms - capture_ts_ms	Logged directly	Not in NPZ
age_ms_reconstructed	Sum of two latency components	Logged in timings	Not in NPZ
Key takeaway: The JSONL logs contain all scalar/statistical metadata but no large tensors. The NPZ files contain the raw tensors (frames, masks, depth maps) but no timing/staleness metadata. The offline tools/visualize_session_replay.py joins these two by trace_id.

Schema Definitions
FramePacket (frame_queue.py):


@dataclass
class FramePacket:
    frame: np.ndarray        # uint8 HxWx3 BGR
    capture_ts_ms: int
    trace_id: str            # UUID hex
VisualizationArtifact (visualization_dump.py):


@dataclass
class VisualizationArtifact:
    trace_id: str
    inference_ts_ms: int
    frame_bgr: np.ndarray    # uint8 HxWx3
    yolo_masks: np.ndarray   # uint8 NxHxW
    class_ids: np.ndarray    # int32 N
    class_names: np.ndarray  # object N
    confidences: np.ndarray  # float32 N
    bboxes: np.ndarray       # float32 Nx4 xyxy
    depth_map: np.ndarray | None  # float16 HxW
